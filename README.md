## Project Description
This project is for reference website for Mining Software Repositories (MSR) researchers.  \
It consists of a main index.html page.  This is the homepage explaining the website.  This homepage has four (4) main links to guide researchers. \
It has the Data Sources link.  This links to a new page of data-sources.html.  This has a table of many links and references to data sources. \
It has the Collection Methods link.  This links to a new page of collection-methods.html.  This has a table linking to many collection methods. \
It has the Analysis Methods link.  This links to a new page of analysis-methods.html.  This has a table referencing many analysis methods. \
It has the Publication Venues link.  This links to a new page of publication-venues.html.  This describes and has a table of many publication venues. \
All four of these linked pages have a Homepage link at the top to direct back to index.html.
